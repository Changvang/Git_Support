Tiếp theo là git remote add origin {cái url git}.

Rồi cái nhánh nào m muốn thì phải làm tiếp như thế này : git rebase origin/{branch_name}.
Cuối cùng là : git push origin master:{branch_name}.
Khuyên là nên thử trước đừng nhào vô cái đùng khéo mất hết bào đó.
